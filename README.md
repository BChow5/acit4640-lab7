# acit4640-lab7


### create ssh key
```ssh-keygen -t ed25519 -f ~/.ssh/aws```
* Generates a new SSH key pair named aws in your ~/.ssh directory.
* -t ed25519 specifies the secure Ed25519

### run script to import and delete key
```./import_lab_key ~/.ssh/aws.pub```
